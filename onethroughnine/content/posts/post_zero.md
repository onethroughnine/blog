## Text Formatting

You can make text **bold** by wrapping it with double asterisks or double underscores. You can make text *italic* by wrapping it with single asterisks or underscores. You can also combine them for ***bold italic*** text.

You can also use ~~strikethrough~~ for deleted text.

## Lists

### Unordered Lists

Unordered lists can be created using hyphens, asterisks, or plus signs:

- First item
- Second item
- Third item
  - Nested item
  - Another nested item

### Ordered Lists

Ordered lists use numbers followed by periods:

1. First item
2. Second item
3. Third item
   1. Nested numbered item
   2. Another nested item

## Links and Images

You can create [inline links](https://example.com) using square brackets for the text and parentheses for the URL.

You can also create reference-style links like [this example][ref], which references a link defined elsewhere.

[ref]: https://example.com

## Code

Inline code can be written using backticks: `const x = 42;`

For code blocks, use triple backticks with an optional language identifier:

```javascript
function hello() {
  console.log("Hello, World!");
  return true;
}
```

```python
def greet():
    print("Hello, World!")
    return True
```

## Blockquotes

Blockquotes are created using the greater-than symbol:

> This is a blockquote. It can span multiple lines and is useful for highlighting important information or citations.
>
> You can even have multiple paragraphs in a blockquote.

Sometimes you need to display special characters literally. Use a backslash to escape them: \*not italic\* or \[not a link\].
